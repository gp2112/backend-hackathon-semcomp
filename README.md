# backend-hackathon-semcomp

## Endpoints

- ``/place``
    - **GET**
    - **Params**:
        - `lat`: *float*
        - `lon`: float

- `/place`
    - **POST**
    - **body**
        lat, lon, name, address, uf, city, created_by, description

- `/team/:id`
    - **GET**

- `/team/:id/matches`
    - **GET**
    - retorna os matches de um time
